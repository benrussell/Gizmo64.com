Documentation for Gizmo64.plugin

Welcome to Gizom64.

Gizmo64 is a plugin for the X-Plane flight simulator that lets you program in Lua Script.

Featuring low level access to access to the simulator and a feature rich API out of the box Gizmo64 is the easiest way to create feature rich content for X-Plane.
